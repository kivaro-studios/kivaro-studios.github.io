# Kivaro Studios

Sito statico multi-page pensato per GitHub Pages.

## Funzionalità incluse

- HTML, CSS e JavaScript separati
- Responsive design desktop / tablet / mobile
- Selettore tema chiaro / scuro con persistenza in localStorage
- Selettore lingua italiano / inglese con persistenza in localStorage
- Pagina progetto dedicata a FlimBox
- Lettura update FlimBox da JSON statici
- Form contatti predisposto per Formspree

## Personalizzazioni rapide

1. Sostituisci `kivaro.studios@gmail.com` con la tua email reale.
2. Sostituisci `your-form-id` in `contact.html` con il tuo endpoint Formspree.
3. Aggiorna `sitemap.xml` e `robots.txt` con il dominio finale.
4. Aggiorna i JSON in `data/flimbox/` quando pubblichi nuove release.
